# Viajes Chile

Este proyecto es una prueba del módulo 2 del curso de desarrollo front end.

## Descripción

Viajes Chile es una plataforma web diseñada para ayudar a los usuarios a planificar y organizar sus viajes a través de Chile. La plataforma ofrece 
información sobre destinos turísticos, opciones de alojamiento, actividades y mucho más.

- **HTML:** Estructura de la página web.
- **CSS:** Estilización y diseño responsivo.
- **JavaScript:** Interactividad y funcionalidad del sitio web.
